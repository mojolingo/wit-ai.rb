# WitAI.rb

A client library for [Wit AI](http://wit.ai).

## Installation

Add this line to your application's Gemfile:

    gem 'wit-ai.rb'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install wit-ai.rb
